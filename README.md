
## 📌 Project Overview

The AI-Based Farmer Predictive System is designed to support National Food Sustainability by predicting the most suitable crop based on environmental conditions.

This system accepts:

- Temperature
- Rainfall
- Soil Type

and provides:

- Recommended Crop
- Sustainability Score
- Sustainability Advice

## ▶️ Installation

Clone repository:

```bash
git clone <repository-link>
```

Go to project folder:

```bash
cd farmer-ai-project
```

Install dependencies:

```bash
pip install flask
```

Run project:

```bash
python app.py
```

Open browser:

```
http://127.0.0.1:5000
```
